## 📌 Overview

This project was developed as part of the **Pattern Recognition & Principles (PRP)** course at IIT Jodhpur.

It focuses on building a **complete Machine Learning pipeline** to predict employee attrition and understand key factors influencing workforce turnover.

> 🎯 Designed to bridge data analysis, machine learning, and business decision-making.

---

## ✨ Key Features

* 🤖 Employee Attrition Prediction Model
* 📊 Data Cleaning & Preprocessing Pipeline
* 📈 Exploratory Data Analysis (EDA)
* 🌳 Decision Tree Model (J48)
* ⚠️ Overfitting & Underfitting Demonstration
* 📊 Advanced Visualizations (Flourish)
* 🧠 AI-driven insights using Napkin AI

---

## 🧠 Project Workflow

The project follows a structured ML lifecycle:

1. 📊 Data Generation (500 records)
2. 🧹 Data Cleaning & Preprocessing (Google Sheets)
3. 📈 Exploratory Data Analysis (Weka)
4. 🌳 Model Building (J48 Decision Tree)
5. ⚠️ Overfitting & Underfitting Analysis
6. 📊 Visualization (Flourish Dashboards)
7. 🧠 AI-driven Insights (Napkin AI)

---

## 📊 Model & Results

* ✅ **J48 Decision Tree Accuracy:** ~94.6%
* ⚠️ **Overfitting Case:** 100% training accuracy → drops to ~87% (cross-validation)
* ❌ **Underfitting (ZeroR):** ~56% accuracy

These results demonstrate the importance of **model generalization** and parameter tuning.

---

## 🔍 Key Insights

* 📉 Low job satisfaction strongly correlates with attrition
* ⏱️ High overtime increases employee exit probability
* 💰 Lower income groups show higher attrition risk
* 🏢 Sales & HR departments show higher attrition trends
